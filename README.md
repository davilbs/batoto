# Batoto - Discord Bot

## Features
### Youtube
- Search songs by URL or search text

### Spotify
- Search tracks by URL or search text
- Play playlists by URL

### Funny
- Generate random names (noun + adjective)

## Heroku deployment
- heroku login 
- git push heroku master 
- heroku ps:scale bot=0 
- heroku ps:scale bot=1
